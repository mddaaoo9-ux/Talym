اختصارات Microsoft Excel

في الصفحة دي هتتعلم أهم اختصارات Excel اللي بتوفر وقتك في التعامل مع الجداول والمعادلات.

الاختصار الوصف نسخ
Ctrl + Nفتح مصنف جديد
Ctrl + Oفتح ملف موجود
Ctrl + Sحفظ المصنف الحالي
Ctrl + Pطباعة المصنف
Ctrl + Cنسخ الخلايا المحددة
Ctrl + Xقص الخلايا المحددة
Ctrl + Vلصق المحتوى المنسوخ
Ctrl + Zتراجع عن آخر عملية
Ctrl + Yإعادة آخر عملية تراجع
Ctrl + ↑الانتقال لأعلى العمود
Ctrl + ↓الانتقال لأسفل العمود
Ctrl + →الانتقال إلى نهاية الصف
Ctrl + ←الانتقال إلى بداية الصف
Ctrl + Shift + Lإضافة أو إزالة الفلاتر
Ctrl + Shift + "نسخ القيمة من الخلية أعلاه
Alt + =إدخال دالة الجمع التلقائي
F2تحرير الخلية المحددة
F4تكرار آخر إجراء
Ctrl + 1فتح إعدادات تنسيق الخلايا
Ctrl + Tتحويل النطاق إلى جدول
